# mmyz · 下一步改进方向(AI版)

> 设计理念：拟真校内网 ARG，玩家有明确身份（牟通雪）和调查动机，叙事清晰可引导。

---

## 🔴 优先级 A — 补强谜题机制

### A1 · 搜索词加密（防 F12 作弊）

**现状问题：** `PINYIN_MAP`、`ID_POSTS`、档案密码均以明文写在 JS 里，玩家 F12 即可看到所有答案。

**改进方案：**

- 将关键词映射表（如论坛 ID 对照、图书馆书目）存入外部 JSON，关键词用 MD5 hash 后存储
- 后端验证改为：`md5(用户输入) === 存储的哈希值`，源代码中不出现原始答案
- 档案密码可保持明文（密码本身就是推理目标），但推导提示需更分散

**预计改动文件：** `js/forum.js`、`js/library.js`、新增 `js/keywords.json`

---

### A2 · 档案密码线索补全

**现状问题：** `GZ2023WOLF` 的提示直接在邮件里写明，三个元素没有分别来自不同页面的线索支撑。

**改进方案：**

- `GZ`（格致）：保持现状，从格致计划名称自然可得
- `2023`：需要在某处新增隐藏线索，例如在 `article_gezhi.html` 中埋入"合作协议签订于2023年"的字样，或 `security.html` 某条记录的年份
- `WOLF`：已有（首页学校简介英文名），保持现状

**预计改动文件：** `article_gezhi.html` 或 `security.html`（添加一行文字即可）

---

## 🟠 优先级 B — 叙事冲击强化

### B1 · 禁词搜索异常页面

**现状问题：** 搜索"林小雨"、"失踪"等高敏词，系统只返回空结果，没有感情表达。

**改进方案：** 搜索特定关键词时，页面出现异常状态：

```
搜索词：「林小雨」
→ 页面弹出：「⚠ 此记录已被教务处清除，如有异议请联系班主任」
→ 底部附：删除时间戳（11月10日 09:03）
```

颜色可短暂变为警告色（橙色边框），强化"有人在删内容"的氛围。

**预计改动文件：** `js/library.js`（图书馆搜索）、`js/security.js`（保安室查询）

---

### B2 · 图书馆借阅记录联动（重要）

**现状问题：** 刘天清借阅的书目（《逆向思维训练》《行为分析导论》等）已在论坛帖子里提到，但图书馆系统里搜索不到，两者没有联动。

**改进方案：**

- 在图书馆系统里，搜索上述书名 → 显示借阅记录，最后借阅人：`ltq2007`，归还日期：**空（未归还）**
- 搜索 `ltq2007` 或 `刘天清` → 显示他所有的借阅历史，包括"11月20日 22:31 入馆，无签出记录"的异常条目
- 这一条目与他在论坛发的最后一帖（22:33）时间完全吻合，形成跨页面推理

**预计改动文件：** `js/library.js`（添加书目数据和借阅记录）

## 🟡 优先级 C — 完善世界细节

### C1 · 树洞（hole）内容丰富化

**现状：** `hole/index.html` 是玩家重要的引导中转点，但匿名帖内容如已比较单薄。

**改进方案：** 在噪音帖中夹带一条关键线索，例如：

```
匿名用户 · 11月16日
旧实验楼那边最近夜里有声音，不是施工声，更像……算了不说了。
反正我不会再晚上去那条路。
```

另可添加几条纯干扰帖（情感倾诉、吐槽老师等），增加"在噪音中找信号"的探索感。

**预计改动文件：** `hole/index.html`

---

### C2 · 时间线一致性对齐

**现状问题：** 各页面时间节点存在轻微矛盾，建议整理后统一。

**参考时间线：**

```
8月          陈昱、校长确认合作
9月          设备到位，志愿者招募帖发出
10月10日    旧实验楼整修公告
10月28日    林小雨首次实验（首批受试者）
11月8日～   论坛热帖讨论外聘顾问
11月10日    教务处发公告压制讨论
11月20日    刘天清颁奖 → 图书馆发现异常 → 被捕获
11月25日    "当前日期"（玩家视角）
```

**预计改动文件：** `index.html`（首页公告日期）、`forum.html`（帖子日期）

---

### C3 · 增加一个"普通学生"视角（可选）

**目的：** 让世界更真实，当前所有可查的 ID 都与案件强相关，缺乏噪音。

**改进方案：** 在论坛固定帖里加一个路过实验楼的普通同学：

```
[分享] 旧实验楼那边晚上怎么有灯
authorId: "ms2009" · 11月12日
下晚自习路过，明明说在整修，灯还是亮着的，还有点声音。
是维修工还是有别的原因？
```

这个帖子搜索该 ID 不会有任何隐藏信息，是纯噪音，但为有心玩家提供一个环境线索。

**预计改动文件：** `js/forum.js`（FIXED_POSTS 追加）


增加仪式感和成就感。

**预计改动文件：** `ending2.html`
