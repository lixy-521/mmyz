# 推送4 格式适配分析 & 插图建议

## 已完成的格式转换

原始 `简要攻略.md` 中有大量微信公众号**不支持或显示异常**的 Markdown 语法，我已全部替换：

| 原格式                    | 问题                                         | 替换方案                                          |
| ------------------------- | -------------------------------------------- | ------------------------------------------------- |
| `# / ## / ###` 多级标题 | 公众号仅支持有限标题样式                     | 改为 `###` 三级标题 + `🔷` emoji 作为段落标题 |
| `> 💡 提示` 引用块      | 深色背景下引用块边框/底色可能不可见          | 直接用 `💡` 前缀的普通段落                      |
| `**加粗**`              | 公众号支持，保留                             | —                                                |
| `1. 2. 3.` 有序列表     | 在公众号编辑器中缩进和间距不稳定             | 改为中文数字"一、二、三"或 `①②③`             |
| `- 无序列表` 嵌套       | 多层级列表在公众号排版极差                   | 改为 `🔹` emoji 前缀，取消嵌套                  |
| Markdown 表格 `\| ... \|` | 公众号**完全不支持**原生 Markdown 表格 | 改为 `🔹 ... → ...` 的平铺文字描述             |
| 代码块 `` `code` ``       | 深色背景下灰底代码块几乎看不到               | 改为普通文字或「」中文引号包裹                    |
| `---` 分隔线            | 公众号支持，保留                             | —                                                |

> [!IMPORTANT]
> 你在公众号编辑器里粘贴时，建议先用 **秀米** 或 **135编辑器** 做排版，然后再同步到公众号，这样可以加背景色块、圆角卡片等效果，深色主题下视觉效果会好很多。

## 关于深色背景的额外建议

> [!TIP]
> 你的公众号是深色（黑色）背景，以下几点需要特别注意：

1. **emoji 选择**：我用了 🔷🔹💡◆ 这几个——它们在深色背景下都有足够的亮度和辨识度。避免使用 ⚫🔲 等暗色 emoji
2. **截图处理**：游戏本身是浅色背景的网页，截图直接放在深色公众号文章里会很突兀。建议：
   - 给截图加 **圆角 + 深色边框/阴影**
   - 或者用排版工具（秀米/135）的"图片卡片"容器来装载
3. **文字颜色**：公众号深色模式下默认文字是浅灰色，一般不用特殊处理。但如果用排版工具，注意不要手动设成深色文字
4. **分隔线**：`---` 在深色背景下可能很暗淡，可以在排版工具里替换为稍亮的分隔线样式
