
提要

这是一个ARG（侵入式）风格网页推理游戏，本文为系列第四期，在看本文之前，请您确保您已经了解第二期（上面链接）的内容。

本文是游戏的简要攻略，如果您追求独立通关这个游戏，请跳转第二期，或者直接点击阅读原文进入游戏。

狼堡迷踪 · 简要攻略

这份攻略给足提示、不给直接答案，让你自己完成每一步，但不会让你无从下手。如果还是卡住了，可以等后续的详细攻略推送。

🔷 游戏大方向

你要做的事情只有三件：

一、把网站逛透——首页、论坛、树洞、师资页面，一个都不能跳过

二、推出四个账号的密码，依次登录——线索全部分散在公开页面里

三、在最后的系统里做一个选择

🔷 第一步：认真看完入场动画

游戏开始时有一段仿微信聊天，是刘天清发给你的消息。

不要急着跳过——他说了一句"如果我消失了……"的话，后半句提到了图书馆和他借过的书，这条线索在游戏后期至关重要。

🔷 第二步：把每个公开页面都看完

进入首页，先把所有新闻都点开读一读。有"↗ 查看全文"的新闻一定要点进去，留意关于「格致计划」的内容——这个词会贯穿整个游戏。

然后进入校园论坛，把固定展示的帖子全部浏览一遍。

💡 有一条教务处发的"公告"帖子——但它的评论区里有人留下了一个新网站的地址，那是接下来最重要的去处。

论坛里还有帖子提到了一位神秘的"陈博士"出现在校园，记一下这个人。

师资团队页面目前只展示校长信息，但上方有个搜索栏——后续你拿到名字时一定要回来搜。

🔷 第三步：逛树洞

从论坛评论区找到树洞地址，进去之后把所有帖子从头到尾读一遍，每一条都要展开看评论。

边读边记下这些东西：

🔹 有帖子在讨论"刘天清不见了"，还有一个女生林小雨参加了旧实验楼的什么活动后也消失了

🔹 有一条帖子附了一张照片——照片里清楚写着一个管理系统的账号和密码

🔹 有一条加密帖，标题叫"把参数反转"——先记住标题，帖子里提示了去论坛搜索什么关键词

🔹 有一条关于图书馆的帖子，"猫头鹰"透露了图书馆系统的账号，还给出了密码的推导方法

🔹 往下翻，"猫头鹰"在更早的帖子里提到了她妈妈的名字和职业——和上面那条对应起来看

🔷 第四步：回论坛，搜刘天清的 ID

树洞加密帖里提示了去论坛搜一个 ID，那就是刘天清。回到论坛搜索那个 ID，读完他发的所有帖子：

🔹 帖①是一道谜题，评论区他亲自确认了答案——这就是加密帖的密码（也是后面推导结局的核心线索）

🔹 帖②提到他最近读了几本书——书名先记下来

🔹 帖④是他消失前的最后一帖，说在借阅系统发现了异常记录

搜完之后，也顺手搜搜树洞和论坛里出现过的那位董老师的 ID——他发了一篇"志愿者招募帖"，地点和要求都很可疑。

💡 董老师的 ID 可以通过师资页面搜到。

🔷 第五步：师资页面查人

进入师资团队，用搜索框查这几个人：

🔹 那位图书馆负责人（猫头鹰帖子里提到了她的名字）

→ 找到她的职务，用职务名称的拼音小写加上 123 就是图书馆密码

🔹 那位理化组老师（搜论坛帖子时看到的那个 ID，师资页能搜到他的详细信息）

→ 记下他的校园网 ID

🔷 第六步：依次登录四个账号

点右上角「🔐 登录」，按顺序来：

◆ 第一个：保安室

账号和密码在树洞那张照片里。

🔹 用监控查询功能查出入记录，特别关注旧实验楼在关键日期的进出人员——留意一个叫"陈昱"的人

🔹 别忘了点侧边栏的「关于本系统」——系统日志里出现了一个高权限账号名，还备注了密码由哪里管理

◆ 第二个：图书馆

账号是猫头鹰帖子里说的那个，密码刚才已经用师资页推出来了。

🔹 分别搜刘天清和林小雨的借阅记录，看看他们各自借了什么书

🔹 切换到「书籍状态查询」，把他们借过的书都查一遍——重点看读者留言

🔹 有一本书的留言里提到某位老师在书里夹了便条，上面有一个隐藏网址

🔹 还有一本书的留言里提到了一种特殊的运算方法——先记下来，这在最后推导结局参数时至关重要

◆ 第三个：教师账号

🔹 在地址栏直接访问留言中提到的那个网址——你会看到一篇私密日记

🔹 日记末尾直接写着这位老师的登录密码

🔹 用这个密码登录教师系统后，把所有邮件从头到尾读完，事件全貌会清晰很多

🔹 其中一封邮件里有实验室档案密码的构成规则——推导出来后解锁档案，记下里面的参数

🔹 去人员档案搜"陈昱"——档案最底部有一个链接，点进去

💡 那个链接会带你去一个外部网站，里面有句话告诉你如何推导最后一个账号的密码。想想项目代号和合作年份。

◆ 第四个：校长

账号名在保安室"关于本系统"的日志里见过，密码从上面的外部网站推导。

🔹 这里有一封来自陈昱的加密邮件——仔细读，提到了一个隐藏的"参数三"，据说是唯一能让受试者自然苏醒的方法

🔹 进入实验室系统，出现了这个游戏最终的操作按钮

🔷 第七步：做出选择

进入实验室系统，搜索「格致计划」。此时系统会要求你输入一个建筑的竣工验收编号才能解锁操作。

💡 怎么找这个编号？ 回想一下你在图书馆查阅书籍留言时，是不是看到过某张图纸档案的线索？拿着那个线索，去图书馆系统的其他功能里查一查。

解锁权限后，你会看到两人的状态，以及最终的操作按钮：

🔹 点「强制结束实验」或输入无效内容 → 结局 0，最坏结局

🔹 输入某个参数（档案里能直接看到的那个） → 结局 1，中性结局

🔹 输入另一个参数（想想刘天清的谜题答案和那本书的留言——对已知参数做那种"运算"） → 结局 2，最好结局

③  树洞的每条帖子都展开看评论了吗？照片仔细看了吗？

④  加密帖的提示跟进了吗？回论坛搜了刘天清的 ID 吗？

⑥  留言里的隐藏网址访问了吗？日记读完了吗？
