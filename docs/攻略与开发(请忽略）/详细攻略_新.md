### 结局 2：绝佳隐藏结局（真结局）——逆向还原
**触发方式**：输入隐藏的参数三数值 **`87.143.212.0`**（不直接展示，必须自主推理得出）。
**推导过程：**
1. 陈昱指出【参数三】能将实验数据原路送回大脑；
2. 刘天清给出的谜题核心答案为 **“逆向”**；

4. 参数基于 IPv4 / 8-bit 的格式，因此我们需将参数二 `168.112.43.255` 每一位对 `255` 取自身补数（即通过 255 减去原数字）：

   - `255 - 112 = 143`
   - `255 - 43 = 212`
   - `255 - 255 = 0`
   合并后解出参数三：**`87.143.212.0`**。

当你输入这个数值，设备顺利执行了深度数据的安全逆转，将意识顺利写回。两人最终转危为安。
