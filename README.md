# 狼堡迷踪

**一个需要你亲自入局的 ARG 风格网页推理游戏**

本说明不含剧透，请放心阅读。

## 游戏简介

深夜十一点，你收到了一条消息：**"如果我消失了，不要相信学校说的任何话。"** 随后，附上了一个再普通不过的校园网链接。

第二天，他的座位空了。学校说他"转学了"。

你登入那个链接，开始在那些学校通知、树洞吐槽和图书馆借阅留言的缝隙里，寻找真相。

## 开始游玩

访问链接即可开始：[https://lixy-521.github.io/mmyz/index.html](https://lixy-521.github.io/mmyz/index.html)

- **纯网页游戏**，无需下载安装任何内容，打开即玩。
- 强烈建议使用 **PC 端浏览器**打开以获得最佳体验（手机端也已做适配）。
- 游戏没有计时器，没有"闯关失败"的判定，你可以随时停下来思考。
- 流程因探索深度而异，约 **30 至 90 分钟**。

## 游玩建议

- **准备好纸和笔。** 线索极其碎片化，可能藏在一则食堂通报里，也可能藏在图书馆的借阅记录中。你的大脑很可能记不住所有东西。
- **如果你发现了两条线索之间意外的呼应，请不要急着往下走。停在那里，待一下。** 那种第一次自己发现线索、仿佛看穿了作者心思的超爽瞬间，正是这个游戏想带给你的最珍贵的体验。

## 内容声明

游戏涉及失踪、信息压制等叙事主题，但无血腥或暴力画面。所有人名、地名、机构名均为虚构。

## 制作说明

本游戏的制作有着 AI 的深度参与。

我们团队负责构建故事的骨架：人物关系、叙事逻辑、线索的埋放方式、谜题的设计方向、校园典故的融合；AI 则将这些化为可以运行的代码——HTML 页面的结构、JavaScript 的交互逻辑、每一个搜索框背后的验证机制。
