# 狼堡一中·学生失踪

**一个关于调查、记忆与代价的网页推理游戏**

---

## 什么是 ARG

**ARG（Alternate Reality Game，替代现实游戏）** 诞生于 2000 年代初期的互联网土壤，它模糊了"游戏"与"现实"之间的边界。玩家不会看到开始界面，没有人告诉你规则，你只是在某个普通的网站上发现了一处说不通的细节，然后沿着这条细缝，越挖越深。

这种设计哲学有一个经典的自我表述：**"This Is Not A Game"——这不是游戏。**

ARG 的叙事通常分散在多个平台与媒介之间，玩家往往需要协作、交叉比对，才能拼出事件的全貌。它最大的魅力不在于谜题本身，而在于那种感觉——你真的在调查一件事，那个世界真的存在。

**几个值得了解的代表作：**

- **Beast（2001）**：迄今公认的第一个大规模 ARG，由微软为电影《人工智能》的宣传而制作。超过 600 个页面藏匿着线索，参与者逾百万。"兔子洞"式入口的概念，由此而来。
- **I Love Bees（2004）**：同样出自微软，为《光环 2》上市造势。一个看似普通的养蜂人网站突然出现数据错乱，牵引出全球玩家协力破解的音频密码。
- **Cicada 3301（2012—2014）**：发布者至今成谜。一系列谜题横跨密码学、古典文学与现实地标，三年间吸引无数高水平参与者，却从未有人宣称通关。
- **大学生登山失踪（国内）**：近年国内民间 ARG 的代表作之一，玩家从一个普通餐馆网站的异常之处入手，逐步揭开一起失踪事件背后的秘密。**本作的创作直接受其启发。**

## 这是什么

这是一个 ARG 风格的**网页推理游戏**。

你将以一名高中生的身份，进入狼堡一中的校园信息网。某位同学突然消失，学校给出的解释滴水不漏，却总有什么地方对不上。游戏没有对话框，没有任务提示，你需要像真正的调查者一样——翻阅论坛，查找档案，读懂那些不该被你看到的邮件，自己把碎片拼成完整的图。

游戏有**三种结局**，由你在最后时刻的选择决定。

---

## 游戏是怎么诞生的

这是一次关于 AI 协作创作的实验。

创作者负责构建故事的骨架：人物关系、叙事逻辑、线索的埋放方式、谜题的设计方向；AI 则将这些化为可以运行的代码——HTML 页面的结构、JavaScript 的交互逻辑、每一个搜索框背后的验证机制。所有的叙事、人物与线索，均为原创虚构内容。

这个项目想探索的，是一个朴素的问题：**一个人，借助 AI，能走多远？**

---

## 游玩方式

- 在浏览器打开链接
- 全程运行于浏览器，无需安装任何软件
- 推荐 **PC 端 Chrome 或 Edge**，体验更佳
- 没有计时器，没有失败判定——你可以随时停下来，想一想
- 建议准备一张纸，随手记下你觉得重要的线索和人名。

---

## 关于内容

游戏涉及失踪、信息压制等叙事主题，但无血腥或暴力画面。所有人名、地名、机构名均为虚构。流程因探索深度而异，约 **30 至 90 分钟**。

停在那里，待一下。那个瞬间，正是这个游戏想给你的。

## 已知局限

这是一次仍在生长中的创作实验，有些地方尚未打磨完整：

- 部分页面在手机端的显示效果未经完整适配
- 游戏尚无存档功能，关闭页面后进度不会保留；但大多数信息是开放可查的，重新调查并不困难
- 内容持续更新中
