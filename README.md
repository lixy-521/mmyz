# 狼堡迷踪

**一个需要你亲自入局的 ARG 风格网页推理游戏**

## 游戏介绍

欢迎登入《狼堡迷踪》。

在这里，你将化身为狼堡一中的一名普通高中生——牟通雪，带着同学消失前留下的神秘代码，登入这所学校的“校园信息网”。

学校给出的官方解释滴水不漏，但只要你足够敏锐，就会发现平静水面下的暗流。

游戏里没有生硬的对话框，也没有手把手的任务提示。你需要像一名真正的黑客或侦探一样——去翻阅看似平常的校园论坛，查找被封存的人员档案，甚至想办法读懂那些绝对不该被你看到的加密邮件，自己把碎片拼成完整的真相。

## 开始游玩与建议

**[👉 点击此处访问链接即可开始游玩](https://lixy-521.github.io/mmyz/index.html)**

本作是一款**纯网页游戏**，无需下载安装任何内容，打开即玩。强烈建议使用 **PC 端浏览器**以获得最佳体验（手机端也已做适配）。游戏没有计时器，也没有“闯关失败”的判定，全流程因探索深度而异，大约需要 **2 个小时**，你可以随时停下来思考。游戏没有计时器，你可以随时停下来思考。游戏共有三种不同的结局，失踪同学的最终命运，完全取决于你最后一刻的选择。

特别提醒，在游玩前请**准备好纸和笔**。游戏中的线索极其碎片化，可能藏在一则食堂通报里，也可能藏在图书馆的借阅记录中，你的大脑很可能记不住所有东西。

如果你发现了两条线索之间意外的呼应，**请不要急着往下走，停在那里待一下**。那种第一次自己发现线索、仿佛看穿了作者心思的超爽瞬间，正是这个游戏想带给你的最珍贵的体验。

> **内容声明**：游戏涉及失踪、信息压制等叙事主题，但无血腥或暴力画面。所有出现的人名、地名、机构名均为虚构。
