# 狼堡一中·学生失踪

**一个关于调查、记忆与代价的网页推理游戏**

本说明不含剧透请放心阅读。

## 零、开始之前

**这是一个在浏览器里运行的网页游戏。** 你不需要安装任何东西，打开链接[https://lixy-521.github.io/mmyz/index.html](https://lixy-521.github.io/mmyz/index.html)就能玩。

请使用PC端打开链接，由于时间和精力限制，我们未对手机端做适配，使用手机游玩会大大降低您的游戏体验以及产生未知bug。

建议你在旁边准备一张纸和笔，随手记下看到的人名、数字和奇怪的细节，因为线索分散在不同的页面里，你的大脑很可能记不住所有东西。

游戏没有计时器，没有"闯关失败"的判定。你可以随时停下来思考，也可以随时从头开始。

## 游戏介绍

这是一个 ARG 风格的**网页推理游戏**（关于ARG的介绍见文档末尾）。

你将以一名高中生的身份，进入狼堡一中的校园信息网。某位同学突然消失，学校给出的解释滴水不漏，却总有什么地方对不上。游戏没有对话框，没有任务提示，你需要像真正的调查者一样——翻阅论坛，查找档案，读懂那些不该被你看到的邮件，自己把碎片拼成完整的图。

游戏有**三种结局**，由你在最后时刻的选择决定。

## 关于内容

游戏涉及失踪、信息压制等叙事主题，但无血腥或暴力画面。所有人名、地名、机构名均为虚构。流程因探索深度而异，约 **30 至 90 分钟**。

## 致玩家

如果你发现了两条线索之间意外的呼应，请不要急着往下走。停在那里，待一下。

那个第一次自己发现线索的感觉，似乎看穿作者心理的超爽瞬间，是这个游戏最珍贵的部分。

祝调查顺利。

## 游戏是怎么诞生的

### 萌芽

我们队伍的三人都来自同一个高中，在进入大学后，我们参加了两次情系母校活动，甚至创建了一个半整活式的公众号“某某yz”。我们会在公众号上分享我们的高中生活经历、学习经验，以及接受其他同学的投稿。此外，我们队伍还一起参加了很多科创比赛，积累了很多合作经验。

在大一阶段，我们就想过能不能利用软件技能给我们的高中母校整点活。但是由于时间和能力的限制一直未能实现。直到大二，也就是现在，随着我们学业的提升和人工智能的突飞猛进，加上拓竹杯软件设计大赛的契机，我们很幸运地又聚在了一起来合作完成一件内心想做的事。

### 成长

本游戏的制作有着AI的深度参与。

我们团队负责构建故事的骨架：人物关系、叙事逻辑、线索的埋放方式、谜题的设计方向、校园典故的融合；AI 则将这些化为可以运行的代码——HTML 页面的结构、JavaScript 的交互逻辑、每一个搜索框背后的验证机制。所有的叙事、人物与线索，均为原创虚构内容。

## 什么是 ARG

**ARG（Alternate Reality Game，替代现实游戏）** 诞生于 2000 年代初期的互联网土壤，它模糊了"游戏"与"现实"之间的边界。玩家不会看到开始界面，没有人告诉你规则，你只是在某个普通的网站上发现了一处说不通的细节，然后沿着这条细缝，越挖越深。

这种设计哲学有一个经典的自我表述：**"This Is Not A Game"——这不是游戏。**

ARG 的叙事通常分散在多个平台与媒介之间，玩家往往需要协作、交叉比对，才能拼出事件的全貌。它最大的魅力不在于谜题本身，而在于那种感觉——你真的在调查一件事，那个世界真的存在。

**几个值得了解的代表作：**

- **Beast（2001）**：迄今公认的第一个大规模 ARG，由微软为电影《人工智能》的宣传而制作。超过 600 个页面藏匿着线索，参与者逾百万。"兔子洞"式入口的概念，由此而来。
- **I Love Bees（2004）**：同样出自微软，为《光环 2》上市造势。一个看似普通的养蜂人网站突然出现数据错乱，牵引出全球玩家协力破解的音频密码。
- **Cicada 3301（2012—2014）**：发布者至今成谜。一系列谜题横跨密码学、古典文学与现实地标，三年间吸引无数高水平参与者，却从未有人宣称通关。
- **大学生登山失踪（国内）**：近年国内民间 ARG 的代表作之一，玩家从一个普通餐馆网站的异常之处入手，逐步揭开一起失踪事件背后的秘密。**本作的创作直接受其启发。**
